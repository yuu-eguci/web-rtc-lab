web-rtc-lab
===

A repository to test WebRTC(real-time-communication) on the GitHub Pages.

- JavaScript からカメラを起動する。
- その映像を、ブラウザ上 video タグに流す。
- iOS では video タグに `autoplay playsinline` 属性が必須であることを明示。
- 映像から画像を取得し base64 にする。
